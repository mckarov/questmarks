# Project Description

## Product summary

QuestMarks is a multilingual tourist exploration app with gamification. The core MVP helps users discover nearby landmarks, check in when they physically reach them, earn XP based on landmark significance, and track their progress through ranks and a shared leaderboard.

## Connection to Criterion A

This implementation follows the Criterion A draft in four main ways:

- It treats mobile architecture, backend integration, and multilingual UX as the main learning goal
- It keeps the motivation rooted in travel, hidden places, family context, and multilingual accessibility
- It turns the planned product into a practical MVP with authentication, geolocation-driven discovery, check-ins, progress tracking, and project presentation
- It preserves the app's relevance as a tool for motivating movement, local discovery, and cultural accessibility

## MVP scope delivered in source

- Authentication flow
- Nearby landmark discovery
- Distance-gated check-ins within 100 meters
- XP and rank progression
- Profile with history and language switching
- Leaderboard
- About screen with project narrative in English, Russian, and German
- Demo runtime for development without Firebase config
