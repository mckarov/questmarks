# QuestMarks Core MVP Release

This release folder packages the first reconstructed MVP of QuestMarks.

## Release contents

- `README.md`: release summary
- `BUILD_AND_RUN.md`: build and bootstrap instructions
- `PROJECT_DESCRIPTION.md`: project narrative derived from Criterion A and repository scope
- `CHANGELOG.md`: high-level implementation summary
- `SOURCE_MANIFEST.txt`: key files included in this release
- `QuestMarks-v1-core-mvp-source.zip`: packaged source snapshot without secrets

## Release intent

The source in the main `QuestMarks` folder now reflects a usable app structure rather than isolated drafts. This release captures the MVP state and the documentation needed to continue development or present the project.
