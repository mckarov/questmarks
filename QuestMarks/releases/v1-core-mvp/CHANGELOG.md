# Changelog

## v1-core-mvp

- Reconstructed QuestMarks into a real React Native application layout with `src/` modules and root app entry files
- Added a full app surface for auth, map discovery, profile, leaderboard, and project overview
- Added lightweight internationalization for English, Russian, and German
- Added a dual runtime model: Firebase when configured, demo mode when not configured
- Preserved and documented the original data utility scripts and seeded landmarks
- Added root documentation plus a release package for handoff and presentation
- Documented the native-shell bootstrap gap because the started project did not include generated React Native platform folders
